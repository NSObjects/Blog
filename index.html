什么是OpenApi规范？ OpenApi规范是一套用语描述REST API的描述格式，分为以下几 func main() { var a = 2 } 描述每个路径上的操作 GET POST DELETE 请求参数和返回值 认证方法 联系信息，许可证，使用条款和其他信息 API规范可以用YAML或JSON编写。该格式对人和机器都很容易学习和阅读。 OpenApi当前版本是3.0，本文将描述OpenApi2.0规范 可以在这里查看完整规范https://github.com/OAI/OpenAPI-Specification/blob/master/versions/2.0.md 本文将用一个YAML格式的例子向你描述如何编写OpenApi2.0文档 文档信息描述 swagger: "2.0" info: title: Sample API description: API description in Markdown. version: 1.0.0 host: api.example.com basePath: /v1 schemes: - https swagger: "2.0" 说明当前使用的OpenApi版本，（Openapi 2.0是用swagger2.0标准演化而来） info对象将描述本文的信息，下表是info对象的所有可用字段 字段 类型 描述 title String 本文标题 description String 本文的描述信息 termsOfService String API的服务条款 contact Object 联系方式对象 license Object 许可证对象 version String 版本 contact 对象用于描述本文作者联系方式
